Class: Box

Box

The `Box` component The main purpose of the `Box` component is to provide a context to the other library components. All other library components should be in the subtree of the `Box` component. They should be placed as the childrens of this component. The description of the context that `Box` component provides is above the declaration of the `Box.childContextTypes` static field. The description of the properties that `Box` component has is above the declaration of the `Box.propTypes` static field.
